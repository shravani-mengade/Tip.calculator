# Tip Calculator

A simple Tip Calculator built using HTML, CSS, and JavaScript that helps users calculate tip amounts and total bill values quickly.

## Features
- Calculate tip amount
- Calculate total bill
- Easy-to-use interface
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Purpose
This project was created to practice JavaScript fundamentals, DOM manipulation, and user input handling.

## How to Run
1. Clone or download the repository.
2. Open `index.html` in your browser.
